# aplikasi-bank-sampah-php

Aplikasi ini gratis, open source, dan siap pakai.Dirancang untuk memudahkan petugas bank sampah dalam mengelola data warga, mencatat transaksi, dan membuat pelaporan. Dilengkapi juga dengan halaman publik yang user-friendly bagi warga untuk mengecek saldo mereka secara mandiri.Aplikasi ini aman, responsif, dan ideal untuk operasional bank sampah di level komunitas, Instalasi (Tinggal Ekstrak & Jalankan)Sangat mudah! Tidak perlu install library manual. Ikuti langkah-langkah berikut untuk menjalankan aplikasi ini di server lokal Anda (misalnya menggunakan XAMPP).1. PrasyaratXAMPP (atau WAMP/MAMP/LAMP) yang sudah terinstal.Pastikan versi PHP 8.0 atau lebih tinggi.Akses ke phpMyAdmin.2. Ekstrak ProyekUnduh file .zip proyek ini.Ekstrak file ZIP tersebut ke dalam direktori htdocs XAMPP Anda.Contoh: C:\xampp\htdocs\aplikasi-bank-sampahSelesai! Folder libs/vendor yang berisi semua library yang diperlukan sudah termasuk di dalam file ZIP. Anda tidak perlu menginstal apa pun lagi.3. Konfigurasi Awal (config/database.php)Ini adalah satu-satunya file yang perlu Anda edit.Buka file config/database.php.Sesuaikan BASE_URL agar sesuai dengan path folder Anda. Ini wajib diubah.// Ganti 'aplikasi-bank-sampah' jika nama folder Anda berbeda
define('BASE_URL', 'http://localhost/aplikasi-bank-sampah/'); 
4. Setup Database (2 Metode)Pilih salah satu metode di bawah ini. Metode Otomatis sangat direkomendasikan.Metode A: Instalasi Otomatis (Rekomendasi)Pastikan file banksampah.sql berada di folder root proyek (sejajar dengan install.php).Buka file install.php di editor kode Anda.Sesuaikan pengaturan koneksi di bagian atas file (biasanya root dan password kosong sudah benar untuk XAMPP).$db_host = 'localhost';
$db_user = 'root'; // User default XAMPP
$db_pass = '';     // Password default XAMPP (kosong)
$db_name = 'db_banksampah';
Buka browser Anda dan akses file installer:http://localhost/aplikasi-bank-sampah/install.phpKlik tombol "Mulai Instalasi Sekarang". Skrip akan membuat database, tabel, dan memasukkan data awal.PENTING! Setelah instalasi berhasil, HAPUS file install.php dari server Anda untuk alasan keamanan.Metode B: Instalasi Manual (Alternatif)Buka phpMyAdmin (http://localhost/phpmyadmin).Buat database baru dengan nama db_banksampah (pastikan collation adalah utf8mb4_general_ci).Pilih database db_banksampah yang baru dibuat.Klik tab Import.Klik "Choose File" dan pilih file banksampah.sql dari dalam folder proyek.Klik "Import" di bagian bawah halaman.5. Selesai!Aplikasi Anda sekarang siap diakses.
